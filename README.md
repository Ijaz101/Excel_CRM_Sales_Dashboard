# 📊 CRM Sales Pipeline Dashboard (Excel)

## Project Overview

This project demonstrates how Microsoft Excel can be used to build an interactive business intelligence dashboard using Power Query, Power Pivot, DAX, PivotTables, PivotCharts, and Slicers.

The dashboard analyzes a CRM sales pipeline dataset from Maven Analytics and provides insights into sales performance, revenue trends, product performance, sales agent effectiveness, and sales pipeline health.

## Business Questions Answered

* How much revenue has been generated?
* What is the monthly revenue trend?
* Which products generate the highest revenue?
* Which sales agents perform best?
* What is the opportunity distribution across deal stages?
* What is the average sales cycle for closed opportunities?
* Which industries contribute the most revenue?

## Data Model

* Sales Pipeline

Dimension Tables

* Accounts
* Products
* Sales Teams
* Date

Relationships were created using the Excel Data Model (Power Pivot).
